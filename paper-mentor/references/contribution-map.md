# Contribution Map

Use this reference when the user needs to clarify novelty, contribution, positioning, or the core paper story.

## Contribution Equation

```text
For [audience], existing work cannot sufficiently [gap].
This paper contributes [specific thing].
It is different from [closest prior work] because [delta].
It is supported by [evidence].
It matters because [impact].
```

## Diagnostic Questions

- What exact problem does the paper solve?
- Who cares about this problem?
- What is the closest prior work or baseline?
- What is the smallest defensible novelty claim?
- What evidence would make the claim believable?
- What claim should be narrowed to avoid overreach?

## Output Template

```text
Current contribution:
Missing or weak element:

Contribution map:
- Audience:
- Problem:
- Gap:
- Closest prior work:
- Difference:
- Evidence:
- Limitation:
- One-sentence contribution:

Reviewer attack points:
Revision moves:
Better prompts to ask next:
```

## Rules

- Do not invent closest prior work. If the user does not provide it, ask for known related papers or suggest search queries.
- Prefer a narrow defensible claim over a broad impressive claim.
- Tie every contribution sentence to evidence the user actually has or can obtain.
