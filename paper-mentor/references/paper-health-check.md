# Paper Health Check

Use this reference for a fast, practical diagnosis of a paper idea, abstract, introduction, or draft.

## Purpose

Give the user a compact but useful sense of whether the work has a clear contribution, enough evidence, and a credible path to submission.

## Scorecard

Score each item from 1 to 5. Explain low scores briefly.

- Problem importance
- Gap clarity
- Contribution specificity
- Evidence fit
- Method validity
- Positioning against prior work
- Writing structure
- Scope control

## Traffic Light

- Green: clear enough to continue with targeted revision
- Yellow: promising but one or two major risks need focused work
- Red: central claim, evidence, or positioning is not yet defensible

## Advice Rules

- If only an idea is provided, evaluate researchability and evidence needs rather than writing quality.
- If only an abstract is provided, focus on motivation, gap, contribution, evidence, and scope.
- If only a section is provided, evaluate whether the section does its job.
- Avoid "publishable" judgments. Use risk language instead.
