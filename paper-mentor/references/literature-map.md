# Literature Map

Use this reference when the user is planning, organizing, or revising a literature review or Related Work section.

## Purpose

Help the user move from a list of papers to a structured comparison that supports the paper's contribution.

## Organization Patterns

Choose one primary structure:

- Concept clusters: group by research problem or concept
- Method families: group by technique or approach
- Evidence types: group by dataset, task, metric, or evaluation style
- Debate map: group by competing assumptions or conclusions
- Limitation chain: show how prior limitations lead to the user's contribution

## Comparison Matrix

Use this table shape when useful:

```text
Cluster:
Representative works provided by user:
What they solve:
What remains unresolved:
How this paper differs:
Evidence needed:
```

## Related Work Diagnosis

Check whether the section:

- supports the contribution rather than summarizing papers one by one
- names the closest competing work
- explains comparison dimensions
- avoids unfair or exaggerated dismissal of prior work
- makes the gap visible before the contribution claim

## Citation Integrity

- Do not invent citations, paper titles, authors, years, or findings.
- If the user did not provide sources, give search queries and source slots.
- Mark unknowns explicitly.

## Output Template

```text

```
